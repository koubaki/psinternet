# Psinternet

Building a truly decentralized network.

## Introduction

The Psinternet is a decentralized network protocol designed to enable secure and efficient communication between nodes without relying on centralized authorities. It utilizes the Node Internetworking Ether (NIE) protocol for transporting PSI (Public System of Internetworking) packets, which are the fundamental units of communication in the Psinternet.

Instead of hierarchical routing, the Psinternet employs a flat routing structure where nodes share paths to other nodes, allowing them to route packets to eachother. Every node in the Psinternet is excepted to have the same path ledger, which is a record of all paths in the network.

The Psinternet is designed to replace the Internet, providing a more secure, efficient, and censorship-resistant communication platform. No government or corporation can control the Psinternet, making it a truly decentralized network for the future.

The Psinternet has no IANA, no CAs, and no ISPs. It is a peer-to-peer network where every node is equal and can communicate directly with each other without intermediaries.

All governments and corporations are called to join the Psinternet and help build a better future for communication. The Psinternet is open-source and anyone can contribute to its development.

Currently, only a few documents exist, but no implementations have been made yet. The Psinternet is still in its early stages, and there is much work to be done to bring it to fruition. However, the vision of a truly decentralized network is worth pursuing, and the Psinternet is a step towards that future.
